## 시작하기

```
# 처음 시작한다면
pnpm install

# 루트 위치에서
turbo run dev
```

### 2. 스토리북 실행

## 기술스택

- Framework: Next.js v15
- Language: TypeScript
- Styling: Tailwind CSS v4
- Build System: Turborepo v2 (Monorepo)
- Package manager: pnpm

## 프로젝트 구조

```
REPO
├─ apps                           # 웹 어플리케이션
│  └─ docs
│  └─ web
├─ packages                       # 공통 UI 라이브러리
│  └─ ui
├─ pnpm-lock.yaml
├─ pnpm-workspace.yaml
└─ turbo.json

```
